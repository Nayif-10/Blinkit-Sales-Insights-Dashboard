# Blinkit-Sales-Insights-Dashboard
# Overview
This project analyzes a sample Blinkit sales dataset using Excel. The workflow includes data cleaning, table formatting, pivot-table analysis, KPI calculation, and dashboard creation. Visuals in the dashboard were generated from .png files included in the repository, and all pivot tables, charts, and datasets are stored in the .xlsx file.

# Workflow

1. Data Cleaning & Preparation

Imported the raw Excel dataset and removed duplicates and invalid entries.
Standardized column names and formatted data types.
Converted cleaned data into structured Excel tables for easy analysis.
