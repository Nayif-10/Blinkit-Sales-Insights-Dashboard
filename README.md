# Blinkit-Sales-Insights-Dashboard
# Overview
This project analyzes a sample Blinkit sales dataset using Excel. The workflow includes data cleaning, table formatting, pivot-table analysis, KPI calculation, and dashboard creation. Visuals in the dashboard were generated from .png files included in the repository, and all pivot tables, charts, and datasets are stored in the .xlsx file.

# Workflow

**1. Data Cleaning & Preparation**
* Imported the raw Excel dataset and removed duplicates and invalid entries.   
* Standardized column names and formatted data types.   
* Converted cleaned data into structured Excel tables for easy analysis.    

**2. KPI Requirements**

* Calculated the following key metrics using pivot tables and formulas:

  * Total Sales

  * Average Sales

  * Number of Items

  * Average Rating

**3. Chart Requirements**

* Created pivot charts to address the required metrics:

  * Total Sales by Fat Content

  * Total Sales by Item Type

  * Fat Content by Outlet (Total Sales)

  * Total Sales by Outlet

  * Sales by Outlet Size

  * Sales by Outlet Location

  * All Metrics by Outlet Type
 
    
**4. Dashboard Creation**

* Combined KPIs and all charts into an interactive Excel dashboard.

* Used .png chart exports for clean visual embedding inside the dashboard.

* Styled the layout for clarity and quick interpretation.

**5. Repository Files**

* BlinkMart_Dashboard.xlsx – Contains the cleaned dataset, pivot tables, dashboards, and all calculations.

* .png – PNG images used for the dashboard’s visual components.
